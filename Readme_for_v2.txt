Описание: 
Telegram-бот, который выдает случайные шутки из баз данных.
Рабочий пример можно найти на канале @Sancho_joke_bot подпишитесь, пожалуйста ^^

Что оно делает
Выдача случайных шуток из баз данных по кнопке;
Добавляйте новые шутки на лету, отправляя их боту.
В один клик узнай id фото

Напиши /admin чтобы пользоваться командами админа

Применение
Получите копию бота

Запустите консоль и введите следущее: *чтобы установить нужные библиотеки:
1) cd *путь к папке*
2) pip install -r requirements.txt

В папку с секретами вставить свой бот-токен и свой Admin-id

Запускаем main.py

( Код сам сгенерирует jokes.db - файл с шутками )
