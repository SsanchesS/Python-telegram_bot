Описание: 
Telegram-бот, который выдает случайные шутки из json.
Рабочий пример можно найти на канале @Sancho_joke_bot подпишитесь, пожалуйста ^^

Что оно делает
Выдача случайных шуток из баз данных по кнопке;

Применение
Получите копию бота

Запустите консоль и введите следущее: *чтобы установить нужные библиотеки:
1) cd *путь к папке*
2) pip install -r requirements_for_v1.txt

В папку с секретами вставить свой бот-токен

Запускаем main.py

( Код сам сгенерирует jokes.json - файл с шутками )
